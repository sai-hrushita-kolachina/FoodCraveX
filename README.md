# FoodCraveX
FoodCraveX is a scalable full-stack food delivery web application designed to simplify online food ordering and management. 
The platform enables users to browse food items and place orders, while providing admins with tools to manage menus and orders efficiently.

## Features
- User-friendly interface for browsing food items and placing orders
- Admin dashboard for managing food items and customer orders
- Secure backend API for handling data and business logic
- Responsive design for all devices

## Tech Stack
- Frontend: React, Vite, HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- Database: MongoDB  
- Deployment: Render



## Installation & Setup
# Clone the repository:
git clone https://github.com/sai-hrushita-kolachina/FoodCraveX.git
cd FoodCraveX

## Backend setup:
cd backend
npm install
npm start

## Frontend setup:
cd frontend
npm install
npm run dev

## Admin panel setup:
cd admin
npm install
npm run dev
